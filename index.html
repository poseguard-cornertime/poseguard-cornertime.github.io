What is PoseGuard?

PoseGuard is an Android cornertime app designed for pose-holding sessions where the user should stay still for a configured amount of time. It uses live camera-based pose tracking to compare the current body position with the initial reference pose.

The app can detect pose drift, sudden movement, and optional face-direction violations. Sessions can include configurable movement sensitivity, voice feedback, optional timer penalties, and timelapse recording.

Offline APK

The Offline APK keeps PoseGuard fully local. It does not request Android INTERNET permission, so Android prevents it from accessing the network.

Online APK

The Online APK adds optional integration with Intiface Central / buttplug.io for compatible external devices. It can send vibration signals while the pose is held or when violations occur.

What PoseGuard checks

At the start of a session, PoseGuard records the initial pose as a reference. During the session, it compares live camera tracking against that reference and detects pose drift, sudden movement, and optional face-direction violations.

PoseGuard is built for Android devices and uses local on-device computer vision instead of uploading camera frames or session data to external servers.

Privacy

Pose and face detection, movement analysis, timer logic, penalties, voice feedback, and timelapse recording are processed locally on the Android device.

PoseGuard does not upload images, video, logs, telemetry, tracking data, session data, or private user data to external servers.

The Online APK uses INTERNET permission only for user-configured integrations such as Intiface Central communication.

Built for offline Android cornertime sessions

PoseGuard is designed as a practical Android cornertime app for sessions where stillness, pose-holding, and movement control matter. It uses the device camera to track the initial pose and compare it with live movement during the session.

The offline version is focused on privacy and local processing. Camera-based pose tracking, movement detection, voice feedback, timer penalties, and timelapse recording run on the Android device without uploading session data to external servers.

The project is open source and available on GitHub: PoseGuard Android cornertime app source code and releases .

Requirements

PoseGuard requires Android 11 or newer and a device with enough performance to process computer vision in real time.
