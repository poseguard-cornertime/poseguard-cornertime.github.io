PoseGuard GitHub

Cornertime and stillness enforcement for Android.

PoseGuard is an Android app for strict pose-holding sessions: cornertime, stillness enforcement, posture control, and consensual D/s or power exchange scenarios where a person must keep a required pose for a configured duration.

Offline

The Offline APK keeps PoseGuard fully local. It does not request Android INTERNET permission, so Android prevents it from accessing the network.

Online

The Online APK adds optional integration with Intiface Central / buttplug.io for compatible external toys. It can send vibration signals while the pose is held or when violations occur.

What PoseGuard checks

At the start of a session, PoseGuard records the initial pose as a reference. During the session, it compares live camera tracking against that reference and detects pose drift, sudden movement, and optional face-direction violations.

Sessions can use configurable duration, movement sensitivity, voice feedback, optional timer penalties, face direction checks, and timelapse recording.

Privacy

Pose and face detection, movement analysis, timer logic, penalties, voice feedback, and timelapse recording are processed locally on the Android device.

PoseGuard does not upload images, video, logs, telemetry, tracking data, session data, or private user data to external servers.

The Online APK uses INTERNET permission only for user-configured integrations such as Intiface Central communication.

Requirements

PoseGuard requires Android 11 or newer and a device with enough performance to process computer vision in real time.
