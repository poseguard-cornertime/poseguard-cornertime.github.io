PoseGuard GitHub

Cornertime app for Android

PoseGuard is a free Android cornertime app for pose-holding sessions. It uses camera-based pose tracking to check whether the user stays still, keeps the initial pose, and follows optional face-direction rules during a timed session.

Latest release: v1.3.1

Pose tracking

How PoseGuard does pose tracking

At the start of a session, PoseGuard records the initial body position as the reference pose. After the timer starts, the app continuously compares live camera tracking against that reference.

This makes PoseGuard different from a simple countdown timer. A normal timer only measures time. PoseGuard also checks whether the pose is actually being held.

Pose drift

PoseGuard detects when the user gradually moves too far away from the starting pose.

Sudden movement

PoseGuard can react to short, sharp movements that break stillness even if the overall pose has not drifted very far.

Optional rule

Face direction

Face tracking can optionally enforce whether the user should face the camera or look away from it. Breaking the selected face-direction rule can be counted as a violation.

Session timer

Timer penalties

PoseGuard can optionally add extra time after violations. This allows a session to continue instead of ending immediately after the first mistake.

Phone setup

How to set up the phone

For reliable tracking, place the Android device in a stable position and keep it still during the session. A moving camera can look like user movement and may cause false violations.

Use portrait orientation and choose a camera angle where the full body is visible. The app works best when the subject fits fully inside the frame before the timer starts.

Good lighting helps the camera and computer vision model see the body silhouette more clearly. Avoid dark rooms, strong backlight, or cluttered backgrounds when possible.

How to choose a pose that tracks well

PoseGuard can be used with different static poses, but tracking works best when important body points are visible. Shoulders, elbows, wrists, hips, knees, and ankles should not be heavily hidden behind the body or outside the frame.

Simple and clearly visible poses are usually more reliable than complex poses where arms or legs overlap heavily. Loose or bulky clothing can make pose tracking harder, while close-fitting clothing usually gives the app a clearer body shape to track.

If you get false violations, try improving lighting, moving the phone farther away, simplifying the pose, lowering sensitivity, or making sure the camera is completely still.

Offline Android cornertime app

The Offline APK is designed for local use. It does not request Android INTERNET permission, so Android prevents it from accessing the network.

Pose detection, face detection, movement analysis, timer logic, penalties, voice feedback, and timelapse recording run on the Android device. The offline version does not upload images, video, logs, telemetry, tracking data, session data, or private user data to external servers.

Online version with optional integrations

The Online APK includes the same core cornertime and pose tracking features as the offline version. It also adds optional integration with Intiface Central / buttplug.io for compatible external devices configured by the user.

Choose the Online APK only if you need network-based integrations. If you want PoseGuard to have no network access at all, choose the Offline APK.

Privacy

Privacy and local processing

PoseGuard is built around local on-device processing. The app does not need an external server to analyze pose, detect movement, run the timer, apply penalties, speak warnings, or record a timelapse.

The Online APK uses INTERNET permission only for user-configured integrations such as Intiface Central communication. Core camera analysis still runs on the Android device.

Who this app is for

PoseGuard is useful when a session needs more than a regular timer. It is made for stillness enforcement, posture training, pose-holding practice, cornertime, and consensual power exchange scenarios where a strict static pose should be held for a configured amount of time.

The app is currently under development, so behavior and available features may change between releases.

Requirements

PoseGuard requires Android 11 or newer and a device with enough performance to process computer vision in real time.

For best results, use a reasonably powerful device, good lighting, a stable phone position, and a camera view where the subject is fully visible.
